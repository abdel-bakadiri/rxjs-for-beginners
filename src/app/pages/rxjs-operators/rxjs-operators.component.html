RxJS operators
of emit values in a sequence and then complete
from create an observable from array or promise
fromEvent create an observable from event
interval emit numbers in sequence based on a period
timer after a delay, emit numbers in sequence based on a period
map turn each emitted value into another value
switchMap map to another observable, complete inner observable and emit values
mergeMap map to another observable and emit values
concatMap map to another observable and emit values in order
filter emit value if it passes a condition
distinctUntilChanged emit value if it's different from previous one
take emit N first values
skip skip N first values and emit others
combineLatest emit latest values from all observables whenever any of them emits a value
forkJoin emit latest values from all observables when all of them completes
zip emit latest values from all observables when all of them emit values
merge emit value whenever any observable emits a value
startWith emit provided value before other values
tap performs any action (side effect)
delay emit values with a delay
timeout throw error if no value is emitted before specified duration
catchError handle error in the stream and replace it with another observable
